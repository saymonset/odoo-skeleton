Descarga masivamente tus documentos del SAT

Este módulo te permite descargar de la página del SAT los XML emitidos y recibidos

Dependencias.

  • El módulo depende de las librerías xmltodict y openssl, para instalar en Ubuntu utiliza los siguientes comandos:
    sudo pip3 install xmltodict
    sudo apt-get install python3-openssl

    • Configuración

  • Agregar el certificado FIEL en la configuración de la compañía


    • Dentro de Facturación / Contabilidad - Ajustes, se pueden realizar configuraciones para el módulo


    • Crear Productos: Permite seleccionar el tipo a utilizar en la creación de productos.
    • Solo documentos de proveedor: Permite descargar solo documentos recibidos.
    • Si producto no se encuentra: En la creación de facturas a partir del XML, se tiene la posibilidad seleccionar si no encuentra un producto lo cree o utilizar un producto ya existente.
    • Buscar por clave SAT: En la busqueda de un producto, permite buscar por la clave SAT.
    • Forma de descarga: API por medio del protocolo de descarga del SAT o WEB utilizando rutinas similares a un navegador. Actualmente solo está funcionando la opción API
    • Tipo conciliación: En la función para concilar documentos, permite seleccionar si debe ser exacta o con rango.
    • Rango +/-: En la conciliación por rango permite establecer rango para la conciliación exitosa del documento.
    • Tipos de descarga


    Uso

    • En el menú de Facturación / Contabilidad agrega una opción llamada ADD con las opciones del módulo:


    • Documentos digitales: Listado de todos los XML descargados.
    • Importar XML: Permite importar XML's de manera manual al listado de XML's.
    • Descargar día: Permite descargar documentos del SAT entre un rango de fechas.
    • Sincronizar SAT: Permite descargar documentos del SAT desde la última descarga hasta el momento actual.
    • Sincronizar documentos: Permite revisar si hay documentos creados en el sistema que coincidan con el UUID de los XML's descargados.
    • Solicitudes WS: Permite revisar las solicitudes de descarga solicitadas al SAT por medio del tipo de descarga API.
    • Listado de documentos XML descargados


    • Al seleccionar uno o varios XML's se tienen varias acciones disponibles


    • Generar facturas: Crea facturas de cliente o proveedor a partir de los XML's.
    • Genera pedidos: Genera ordenes de compra o pedidos de venta a partir de los XML's.
    • Conciliar: Coloca la información fiscal a una factura a partir de un XML.
    • Desconciliar: Quita la información fiscal agregada en la conciliación del documento.
    • Exportar XML: Descarga los archivos XML en un archivo zip.
    • Desargar estado del SAT: Descarga el estado del XML en el SAT.
    • Leer información adiiconal XML: Lee campos adicionales del XML y los agrega en la vista de lista de los XML's
    • Genera pólizas: Genera una póliza apartir de los XML's de facturas de cliente o proveedor.
    • Genera Gasto: Genera un reporte de gastos con una línea de gasto por cada producto en el XML
    • Listado de solicitudes de WS


  • El proceso de descarga API sigue un proceso: Solicitud, Validación y Descarga. Dentro de los documentos solicitudes WS se puede ver el detalle del proceso y su resultado.
