Overview

This module enhances the "Category Pictures" feature in Odoo POS by introducing a horizontal slider for product categories. Instead of a standard scroll bar, the "Category Slider" allows you to scroll through categories with pictures seamlessly. Simply enable "Category Pictures" and "Category Slider" to enjoy a smoother, more intuitive browsing experience in your POS interface.

Features

Seamlessly manage product categories with an intuitive slider in your POS interface.

Effortlessly browse subcategories using the same slider for a smooth user experience.

No additional configuration required—simply install the module and start using it immediately.

Configuration

  • As shown in the image below, the default Odoo POS displays a vertical scrollbar for product categories when "Category Pictures" is enabled, making navigation less intuitive when there are many categories. This module replaces the traditional scrollbar with a sleek horizontal slider, providing a smoother and more user-friendly experience for browsing product categories directly in the POS interface.
