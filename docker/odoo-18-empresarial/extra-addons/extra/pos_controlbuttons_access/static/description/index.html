Advanced POS Hide Control Buttons User Access

This Odoo POS module enhances user interface customization in the Point of Sale (POS) system by allowing different user roles to hide specific buttons and actions based on their permissions. This aims to streamline the POS interface for users, enabling them to focus on the actions relevant to their tasks.



Features
  • POS Hide-Show Actions Button.
  • POS Hide-Show Customer Button.
  • POS Hide-Show Internal Note Button.
  • POS Hide-Show Quotation/Order Button.
  • POS Hide-Show Tax Button.
  • POS Hide-Show Refund Button.
  • POS Hide-Show General Note.
  • POS Hide-Show Customer Note.
  • POS Hide-Show Pricelist Button.
  • POS Hide-Show Cancel Order Button.
  • Support Community and Enterprise Editions
